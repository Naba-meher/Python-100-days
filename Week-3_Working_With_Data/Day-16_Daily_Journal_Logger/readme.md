# Day 16 - Daily Journal Logger

## Objective
Learn file writing in Python by building a simple journal application that stores and searches daily entries.

## Features
- Add journal entries
- View all entries
- Search entries by keyword
- Save entries permanently in a text file
- Handle file-related errors

## Concepts
- File Writing (`w`)
- File Appending (`a`)
- File Reading (`r`)
- Exception Handling
- Functions

## Run

```bash
python main.py
```
