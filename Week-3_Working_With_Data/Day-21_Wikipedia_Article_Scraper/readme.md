# Day 21 - Wikipedia Article Scraper

## Objective
Learn web scraping in Python by extracting information from Wikipedia using the `requests` and `BeautifulSoup` libraries.

## Features
- Search Wikipedia articles by topic
- Extract article title
- Display the article summary
- List article headings
- Fetch related Wikipedia links
- Handle invalid topics

## Concepts
- Web Scraping
- HTML Structure
- `requests`
- BeautifulSoup
- HTML Parsing

## Run

```bash
python main.py
```

## Requirements

```bash
pip install requests beautifulsoup4
```
