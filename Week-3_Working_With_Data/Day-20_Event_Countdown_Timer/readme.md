# Day 20 - Event Countdown Timer

## Objective
Learn how to work with Python's `datetime` module by building an event countdown timer.

## Features
- Accept an event date and time from the user
- Calculate the remaining time until the event
- Display a live countdown
- Validate date and time input
- Notify when the countdown is complete

## Concepts
- `datetime` Module
- Date & Time
- Date Formatting
- Time Difference Calculation
- `time.sleep()`

## Run

```bash
python main.py
```
