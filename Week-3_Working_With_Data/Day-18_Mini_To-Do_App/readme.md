# Day 18 - Mini To-Do App

## Objective
Learn how to store, read, update, and manage data using JSON files by building a simple To-Do application.

## Features
- Add new tasks
- View all tasks
- Update task status
- Delete tasks
- Store tasks permanently in a JSON file

## Concepts
- JSON Files
- `json.load()`
- `json.dump()`
- File Handling
- Functions
- Exception Handling

## Run

```bash
python main.py
```
