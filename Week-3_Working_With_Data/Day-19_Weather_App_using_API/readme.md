# Day 19 - Weather App using API

## Objective
Learn how to fetch real-time weather data from an API using Python and the `requests` library.

## Features
- Get live weather information by city
- Display temperature, humidity, weather condition, and wind speed
- Handle invalid city names
- Fetch data using the OpenWeatherMap API

## Concepts
- APIs
- HTTP Requests
- API Keys
- `requests` Library
- JSON Responses
- Exception Handling

## Run

```bash
python main.py
```

## Requirements

```bash
pip install requests
```
