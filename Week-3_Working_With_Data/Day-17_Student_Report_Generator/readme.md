# Day 17 - Student Report Generator

## Objective
Learn how to read, process, and write CSV files using Python's `csv` module.

## Features
- Read student records from a CSV file
- Calculate average marks
- Assign Pass/Fail status
- Generate a new CSV report
- Handle file and data errors

## Concepts
- CSV Files
- `csv.reader()`
- `csv.DictReader()`
- `csv.writer()`
- `csv.DictWriter()`
- Exception Handling

## Run

```bash
python main.py
```
