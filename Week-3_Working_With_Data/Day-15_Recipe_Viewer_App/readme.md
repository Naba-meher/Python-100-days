# Day 15 - Recipe Viewer App

## Objective
Learn file reading in Python by building a recipe viewer that loads and displays recipes from a text file.

## Features
- Read recipes from a text file
- View a recipe by name
- List all available recipes
- Handle missing file errors gracefully

## Concepts
- File Reading
- `open()`
- File Modes (`r`, `rb`, `r+`)
- Exception Handling
- Dictionaries
- Functions

## Run

```bash
python main.py
```
