# Day 2 Notes

## Learned
- Variables store values.
- Python supports `str`, `int`, `float`, and `bool`.
- `input()` always returns a string.
- Use `int()` to convert input to an integer.
- f-strings make output clean and readable.

## Key Syntax

```python
name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Hello {name}, you are {age} years old.")
```

## Takeaway
Variables and type conversion help us work with different kinds of user input efficiently.
