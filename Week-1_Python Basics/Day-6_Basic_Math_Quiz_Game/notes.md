# Day 6 Notes

## Learned
- Functions group reusable code.
- Functions are defined using `def`.
- Parameters receive values passed as arguments.
- `return` sends a value back to the caller.

## Function Syntax

```python
def greet(name):
    print(f"Hello, {name}!")

greet("Naba")
```

## Function with Return

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```

## Key Terms

| Term | Meaning |
|------|---------|
| Function | A reusable block of code |
| Parameter | Variable in a function definition |
| Argument | Value passed to a function |
| Return | Sends a value back from a function |

## Takeaway
Functions make code modular, reusable, and easier to maintain. Use `return` when a function needs to provide a result.
