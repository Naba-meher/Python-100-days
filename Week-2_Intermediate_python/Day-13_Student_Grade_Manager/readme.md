# Day 13 - Student Grade Manager

## Objective
Learn how to use list comprehensions to assign grades and filter student scores.

## Features
- Accept multiple student scores
- Assign grades (A–F) using list comprehensions
- Display each student's score and grade
- Filter passing and failing students

## Concepts
- List Comprehensions
- Conditional Expressions
- Filtering Lists
- `enumerate()`
- `zip()`

## Run

```bash
python main.py
```
