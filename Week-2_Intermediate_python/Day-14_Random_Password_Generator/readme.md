# Day 14 - Random Password Generator

## Objective
Learn how to use Python modules and libraries by building a random password generator.

## Features
- Generate secure random passwords
- User-defined password length
- Includes uppercase, lowercase, digits, and special characters
- Password length validation using exceptions

## Concepts
- Modules & Libraries
- `import` Statements
- Built-in Modules (`random`, `string`, `math`, `datetime`, `os`)
- Custom Modules
- Exception Handling

## Run

```bash
python main.py
```
