# 📒 Contact Book

A command-line Contact Book application built with Python as part of my **100 Days of Python** journey.

## ✨ Features

- ➕ Add new contacts
- 📋 View all contacts
- 🔍 Search contacts by name
- ✏️ Edit existing contacts
- 🗑️ Delete contacts
- 📊 Display total number of contacts
- 💾 Automatically save contacts to a JSON file
- 📂 Load existing contacts when the application starts
- ✅ Phone number validation (10 digits)

## 🛠️ Technologies Used

- Python
- Dictionaries
- Functions
- JSON File Handling
- Loops & Conditionals
