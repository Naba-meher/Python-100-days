# Day 12 - Temperature Converter

## Objective
Build a menu-driven temperature converter using Python functions and return values.

## Features
- Celsius → Fahrenheit & Kelvin
- Fahrenheit → Celsius & Kelvin
- Kelvin → Celsius & Fahrenheit
- Menu-driven interface
- Formatted output (2 decimal places)

## Concepts
- Functions
- Return Values
- Multiple Functions
- Arithmetic Operations
- User Input
- f-Strings

## Run

```bash
python main.py
```
